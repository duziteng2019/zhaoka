# 证件照小程序 - 产品需求文档

## Overview
- **Summary**: 证件照制作微信小程序 MVP，提供手机拍照、AI 自动裁剪、背景替换、标准证件照导出等功能。用户可以通过小程序快速制作符合规格要求的证件照。
- **Purpose**: 解决用户快速制作证件照的需求，提供便捷的证件照制作工具，支持多种规格和背景替换。
- **Target Users**: 需要制作证件照的微信用户，包括学生、职场人士、出国人员等。

## Goals
- 提供便捷的证件照制作流程
- 支持常用证件照规格（一寸、二寸、护照、签证）
- 实现 AI 自动裁剪和背景替换功能
- 提供免费低清下载和付费高清下载选项
- 支持历史记录查看与重新编辑

## Non-Goals (Out of Scope)
- 趣味模板功能
- 社交分享功能
- 提醒通知功能
- 更多证件照规格（如各国签证特殊规格）

## Background & Context
- 项目使用微信小程序原生开发，基于云开发架构
- 集成腾讯云 AI 服务进行人脸检测和人像分割
- 使用云开发数据库和云存储进行数据管理
- 采用 Editorial/Magazine 设计风格

## Functional Requirements
- **FR-1**: 首页展示 - 显示证件照制作快捷入口和常用规格
- **FR-2**: 照片上传 - 支持拍照和相册选择两种方式
- **FR-3**: 规格选择 - 提供一寸、二寸、护照、签证四种规格
- **FR-4**: AI 自动裁剪与手动微调 - 自动识别人脸并生成裁剪框
- **FR-5**: 背景替换 - 支持白、蓝、红三种背景颜色替换
- **FR-6**: 导出与支付 - 提供免费低清下载和付费高清下载
- **FR-7**: 历史记录查看 - 展示用户所有历史制作的证件照
- **FR-8**: 历史详情与重新编辑 - 查看历史证件照详情并支持重新编辑
- **FR-9**: 用户身份管理 - 自动获取用户 OPENID 实现天然免登录

## Non-Functional Requirements
- **NFR-1**: 性能 - 页面加载时间不超过 3 秒，图片处理响应时间不超过 5 秒
- **NFR-2**: 安全性 - 数据库权限控制，用户只能访问自己的数据
- **NFR-3**: 可用性 - 支持 iOS 和 Android 微信客户端，适配不同屏幕尺寸
- **NFR-4**: 用户体验 - 界面美观，操作流程清晰，反馈及时
- **NFR-5**: 可扩展性 - 系统架构支持后续功能扩展

## Constraints
- **Technical**: 基于微信小程序平台，使用云开发服务
- **Business**: MVP 阶段重点实现核心功能，控制开发成本
- **Dependencies**: 依赖腾讯云 AI 服务，需要配置 API Key

## Assumptions
- 用户拥有微信账号并安装最新版本微信客户端
- 用户授权小程序使用摄像头和相册权限
- 云开发环境已正确配置
- 腾讯云 AI 服务 API 密钥已正确配置

## Acceptance Criteria

### AC-1: 首页展示
- **Given**: 用户打开小程序
- **When**: 进入首页
- **Then**: 显示证件照制作快捷入口和立即制作按钮，展示常用规格（一寸、二寸、护照、签证）
- **Verification**: `human-judgment`
- **Notes**: 界面采用 Editorial/Magazine 风格，使用暖橙红色主色调

### AC-2: 照片上传
- **Given**: 用户进入上传照片页面
- **When**: 选择拍照或相册选择
- **Then**: 成功上传照片并预览，显示下一步按钮
- **Verification**: `programmatic`
- **Notes**: 上传失败时应提示用户重新上传

### AC-3: 规格选择
- **Given**: 用户进入规格选择页面
- **When**: 选择不同规格
- **Then**: 系统配置对应规格参数并跳转到裁剪调整页面
- **Verification**: `programmatic`
- **Notes**: 一寸(25×35mm)、二寸(35×49mm)、护照(33×48mm)、签证(35×45mm)

### AC-4: AI 自动裁剪
- **Given**: 用户进入裁剪调整页面
- **When**: 系统调用 AI 人脸检测
- **Then**: 自动生成裁剪框，用户可手动微调
- **Verification**: `programmatic`
- **Notes**: AI 无法检测人脸时应提示用户手动调整

### AC-5: 背景替换
- **Given**: 用户进入背景替换页面
- **When**: 选择背景颜色并调整美颜效果
- **Then**: 实时更新预览效果，生成最终图片
- **Verification**: `programmatic`
- **Notes**: 支持白、蓝、红三种背景颜色

### AC-6: 导出与支付
- **Given**: 用户进入导出页面
- **When**: 选择免费下载或付费高清下载
- **Then**: 成功下载对应分辨率的证件照
- **Verification**: `programmatic`
- **Notes**: 免费版本 800×1200 像素，高清版本 2000×3000 像素

### AC-7: 历史记录查看
- **Given**: 用户进入历史记录页面
- **When**: 浏览历史记录
- **Then**: 展示所有历史制作的证件照，按创建时间倒序排列
- **Verification**: `programmatic`
- **Notes**: 显示每张证件照的缩略图、规格、创建时间

### AC-8: 历史详情与重新编辑
- **Given**: 用户进入历史详情页面
- **When**: 点击重新编辑按钮
- **Then**: 加载原始照片并跳转到裁剪调整页面
- **Verification**: `programmatic`
- **Notes**: 重新编辑后应创建新的历史记录

### AC-9: 用户身份管理
- **Given**: 用户首次使用小程序
- **When**: 系统自动获取 OPENID
- **Then**: 关联用户数据和订单记录，实现天然免登录
- **Verification**: `programmatic`
- **Notes**: 不需要显式的登录流程
